# DocTalk

DocTalk is a doctor discovery and appointment platform that helps patients find the right specialists, book visits quickly, and manage their care in one place. It also includes an analytics view that highlights which doctors receive the highest payments based on appointment activity.

## Highlights

- Doctor discovery with profile cards, specialties, and availability.
- Appointment booking flow with confirmation and booking history.
- Patient dashboard for managing upcoming and past appointments.
- Analytics chart showing top-earning doctors based on booked appointments.
- Responsive UI with a mobile-friendly navigation menu and quick actions.

## Pages

- Home
- Doctors Directory
- Doctor Details
- My Bookings
- Blogs
- Contact Us
- Analytics (Top Earnings)

## Tech Stack

- React + Vite
- React Router
- Tailwind CSS
- DaisyUI

```bash
npm install
```
